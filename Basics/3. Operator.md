# **Arithmetic Operators :**

7 types.

### **Binary Operators :**  

1. `Addition + : `  
      eg. 6 + 3 = 9  
          7.3 + 32.4 = 39.7  
            
2. `Subraction - :`  
      eg. 8 - 7 = 1  
          2.3 - 5.4 = - 2.1  
          
3. `Multiplication * :`  
      eg. 4  * 9 = 36  
          2.5 * 2.0 = 5.0  
          
4. `Division / :`  
      eg. 7 / 2 = 3.5  
          8.5 / 2.0 = 4.25  
          
5. `Floor division // :`  
      eg. 7 // 2 = 3  
          7.0 // 2 = 3.0
         
6. `Modulus % :`  
      eg. 8 % 3 = 2  
          
7. `Exponential ** :`  
      eg. 2 ** 3 = 2^3 = 8  
          2.0 ** 3 = 8.0

___

### **Unary Operators :**  

1. `Plus + :`  
      eg. +4, +3  
      
3. `Minus - :`  
      eg. -74, -32
