# **Literals :**  

### Literal Types:
 1. Integer   
      eg. `7`, `-554`, `0`.
      
    * Octal :
           Starts with `0o`.  
           eg. `0o232` - 2 * 8^2 + 3 * 8^1 + 2 * 8^0
               0o232 - 154
    
    * Hexadecimal :
            Starts with `0x`.  
            eg. `0x123` - 1 * 16^2 + 2 * 16^1 + 3 * 16^0
                0x123 - 291
 
 2. Floating pt  
       eg. `58.25`, `-8402.8784` , `1e - 22`.
 
 3. Strings  
        eg. `"hi"`, `'hello'`, `"sup'"`, `'that "it"'`.
> **Hint :**  
      We use \ in order to print the string itself.  
      ```python
      print("Man was and is an \"animal\".")
      ```  
      O/P:  
          Man was and is an "animal".  
      
 4. Boolean  
        `True` or `False`.  
        True -> 1.  
        False -> 0.
