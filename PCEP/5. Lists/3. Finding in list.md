# **Finding in list :**  

We use `in` to check if an element is in the list.  

eg.   
```python
if "abc" in list :
    print("abc")
else:
    print("123")
```
___ 


Similarly `not in` is used to check if an element is not in the list.  

eg.     
```python
if "abc" not in list :
    print("123")
else:
    print("xyz")
```
