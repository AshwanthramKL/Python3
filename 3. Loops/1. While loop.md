# **While loop :**

Execute a code block repeatedly until a condition is satisfied.  
    
*Syntax:*  

```python
while condition:
    Statements
        .
        .
        .
```

**[ OR ]**

```python
while condition:
    Statements
        .
        .
        .
else:
    Statements
        .
        .
        .
```  
Else statement is not compulsory.  
___

**__eg.__**   
```python
i = 1
while i < 6:
  print(i)
  i += 1
else:
  print("i is no longer less than 6")
```

```
1  
2

4  
5
```
